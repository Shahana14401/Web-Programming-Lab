Selectors in CSS

A CSS selector selects the HTML element(s) for styling purpose. CSS selectors select HTML elements according to its id, class, type, attribute etc.

Id selector: The id selector uses the id attribute of an HTML element to select a specific element. Note: An id of element is unique on a page to use id selector.

Class-selector

The class selector selects HTML elements with a specific class attribute.
To use class selector you must use ( . ) followed by class name in CSS.
